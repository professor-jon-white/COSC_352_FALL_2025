## Overview

This project uses **R** and **RShiny** to scrape, process, and visualize Baltimore City homicide data from 2021 to 2025. The dashboard provides:

- Yearly homicide counts
- Age distribution of victims
- Interactive map of incidents
- Searchable and filterable data table

The goal is to make the data **easy to explore and analyze** for trends and patterns.
